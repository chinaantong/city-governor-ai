# 城市决策者 (City Governor AI)

一个结合**城市模拟**与**AI决策对话**的策略游戏。玩家作为市长，通过任命副市长、设置政府部门、听取AI建议，做出影响城市命运的决策。

## 🎮 游戏核心特性

### 城市建筑系统
- 参考《天际线2》的城市管理机制
- 住宅、商业、工业、绿地区域规划
- 基础设施建设（道路、电力、水系、污水）
- 资源生产与消耗循环

### 政府部门结构
- **7个核心部门**：交通、环保、教育、卫生、城建、警察、财政
- **副市长系统**：1-5名副市长管理不同部门组合
- **市议会机制**：大额预算和重大决策需投票表决
- **动态定制**：玩家可创建新部门、调整权限

### AI决策引擎
- **有性格的AI副市长**：每个副市长有独立立场和沟通风格
- **主动建议系统**：AI会主动分析问题、提出方案
- **AI学习机制**：系统学习玩家偏好，调整建议策略
- **角色扮演对话**：支持菜单选择和自由文本对话

### 城市危机事件
- 污染爆发、交通瘫痪、经济衰退、犯罪浪潮、自然灾害、疫情爆发
- 每个危机有多个应对方案，选择影响城市发展
- 可在新手模式关闭

### 游戏失败条件
- ❌ 市民满意度持续低于30%，被市议会罢免
- ❌ 城市负债超过年收入200%，城市破产
- ❌ 人口离开超过50%或失业率>40%，城市崩溃
